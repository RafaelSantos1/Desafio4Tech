## Executar pelo Visual Studio

1. Abra o arquivo Desafio4Tech.sln no Visual Studio.

2. Defina Desafio4Tech.API como projeto de inicialização.

3. Clique em Iniciar (F5) para executar a API.

## Como Executar pelo VsCode 

### 1. Build do projeto
dotnet build Desafio4Tech.sln

### 2. Executar a API
dotnet run --project Desafio4Tech.API/Desafio4Tech.API.csproj

## A API estará disponível em:
https://localhost:44307
